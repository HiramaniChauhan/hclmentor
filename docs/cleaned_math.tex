\subsection*{Quadratic Equation}
Solve the quadratic equation
\[2x^{2}-4x-6=0.\]
Coefficients: $a=2$, $b=-4$, $c=-6$. Discriminant
\[\Delta=b^{2}-4ac=(-4)^{2}-4(2)(-6)=16+48=64.\]
By the quadratic formula
\[x=\frac{-b\pm\sqrt{\Delta}}{2a}=\frac{-(-4)\pm\sqrt{64}}{4}=\frac{4\pm8}{4},\]
so
\[\boxed{x=3\quad\text{or}\quad x=-1.}\]

\subsection*{Basic Integrals}
\[\int(3x^{2}-2x+1)\,dx = x^{3}-x^{2}+x+C.\]
\[\int(3x^{2}+2x+1)\,dx = x^{3}+x^{2}+x+C.\]

\subsection*{Power Rule}
For $n\neq -1$,
\[\int x^{n}\,dx=\frac{x^{n+1}}{n+1}+C.\]
Applying to $3x^{2}$:
\[\int 3x^{2}\,dx = 3\int x^{2}\,dx = 3\cdot\frac{x^{3}}{3}=x^{3}+C.\]

\subsection*{Integration by Parts Example}
Compute $I=\int e^{x}\cos x\,dx$.
Choose $u=e^{x}$, $dv=\cos x\,dx$ so $du=e^{x}\,dx$, $v=\sin x$. Then
\[I=e^{x}\sin x-\int e^{x}\sin x\,dx.\]
Let $J=\int e^{x}\sin x\,dx$. Integrate $J$ by parts with $u=e^{x}$, $dv=\sin x\,dx$:
\[J=-e^{x}\cos x+\int e^{x}\cos x\,dx = -e^{x}\cos x + I.\]
Substitute back:
\[I=e^{x}\sin x - J = e^{x}\sin x -(-e^{x}\cos x + I) = e^{x}(\sin x + \cos x) - I.\]
Thus
\[2I = e^{x}(\sin x + \cos x)\quad\Rightarrow\quad I=\frac{e^{x}(\sin x + \cos x)}{2} + C.\]

\subsection*{Integral of $\sec^{3}\theta$ (useful identity)}
The integral
\[\int \sec^{3}\theta\,d\theta = \frac{1}{2}\sec\theta\tan\theta + \frac{1}{2}\ln|\sec\theta + \tan\theta| + C.\]
(This result may be derived by integration by parts or using the identity $\tan^{2}\theta=\sec^{2}\theta-1$ and splitting the integrand.)
